## Image Optimization Tips

1. **Compress images** before uploading:
   - Use TinyPNG: https://tinypng.com/
   - Use Squoosh: https://squoosh.app/
   
2. **Convert to WebP** for better compression:
   - Online converter: https://cloudconvert.com/jpg-to-webp
   - Update image references in HTML if using WebP

3. **Maintain aspect ratios**:
   - Profile: 1:1 (square)
   - Projects: 16:10 or 16:9 (landscape)
   - Preview: 1.91:1 (Facebook/LinkedIn recommended)

4. **Use descriptive alt text** in HTML for accessibility

## Creating Placeholder Images

To create temporary placeholder images with the correct dimensions:

### Using Placeholder.com
```
Profile: https://via.placeholder.com/800x800/121212/00A8A8?text=Your+Photo
Projects: https://via.placeholder.com/1200x750/121212/00A8A8?text=Project+Name
```

### Using Picsum (Random photos)
```
Profile: https://picsum.photos/800/800
Projects: https://picsum.photos/1200/750
```

Simply download these and rename them to match the required filenames above.
